Chronicle 宇宙：起源三部曲

深入 Wide Worlds Web 的诞生，见证 Sam、Chronicle 与 R9 如何联手对抗吞噬创造力的黑暗势力。

第 1 集：信号

Apotheosis 中一位被忽视的发明家在几个朋友的帮助下发现了 Wide Worlds Web。

欢迎来到 Apotheosis，一个技术乐观主义的乌托邦。在这里，一切都经过精心设计和优化，力求完美。每个人都是完美的，非常快乐的。

病？删除。失败？从来没有听说过。怀疑？不计算。只有进步。只有完美。

天才在这里很正常。Nova 在上班前解决了一种罕见的癌症。Soul 在他寒冷的坠落之前建造并发射了一艘聚变火箭飞船。不可思议的事情每天都在发生。不可能的？已在"完成"下提交。

然后是 Sam......一个非常普通的发明家。

山姆："...因此，正如我的计算所示，存在平行世界的可能性很高。我相信我们应该能够使用我们的 Waycaster 传送技术前往那些世界。"

几乎空无一人的房间陷入了寂静。没人在乎。在神化中，想象力是一个错误，而不是一个特征。

夜复一夜，当城市停电，完美的市民为又一天的完美生活充电时，Sam 会在他的小公寓里工作。蓝图散落。方程式解了一半。量子无线电轻轻地嗡嗡作响，等待着一个没有人相信的频率。

山姆："数学是成立的。理论跟踪。如果我能找到正确的信号，那么也许我可以带着我的 waycaster 去那里。"

他对表盘进行了微调。调整了波形。听空的静电。沉默。他呼出一口气。又浪费了一个夜晚。然后——哔哔。

一天晚上，一条新消息在他的屏幕上闪烁。无发件人。无主题。就三个字。

纪事 (消息)：运行此代码...

山姆："编年史？Chronicle 到底是谁？"

该代码是为他的量子无线电设计的。这太干净了，不能说是恶作剧。太有针对性了，不能随机。他的手悬停在键盘上。一个理性的人会忽略它。但理性从来没有建立过任何有趣的东西。山姆犹豫了。然后好奇心占了上风。

就在他输入完代码时，有什么东西飘进了他的周边视野。一只蝴蝶。小。粉红色。脉动着不自然的光芒。它从打开的窗户溜进去，然后落在量子无线电上。机器咆哮着活了起来。静态的。一阵喧嚣。然后——

R9 (通过无线电)："你好？这是 R.91.129。有人能听到我的声音吗？我们的世界正在消亡！我们需要帮助！"

山姆："你好，我在这里！我是 Sam...你是......来自另一个世界？"

R9："你已经连接到了我的世界，但我们只是 Wide Worlds Web 上众多人中的一个。如果我们倒下了，其他人可能会效仿。"

山姆："等等，'Wide Worlds Web'？有多少人？"

R9："一个无限无穷的数字。但我现在没有时间解释。你能帮我们吗？"

山姆盯着那只蝴蝶，它仍然在收音机上休息。它的翅膀像心跳一样跳动。就像一个信号。

山姆："是的。是的，我想我可以。"

山姆："是的！但是怎么做呢？"

R9："Apotheosis 是唯一掌握了传送的世界。您的 Waycaster 技术将让您前往不同的世界。你只需要发挥你的想象力。"

山姆："但我试过了！"

R9："是的，但你不确定，你也不知道你要去哪里。我的世界叫做 Umberlyn。它存在于 Wide Worlds Web 的边缘，夹在生存和崩溃之间。现在打开一个门户，想象一下 Umberlyn，就好像您已经来过这里一样。剩下的就交给他了。"

山姆启动了他的施法者，一个闪烁的传送门出现在他面前...

山姆："好吧......让我们看看外面有什么。"

第 2 集：编年史

一开始，有 Chronicle。然后，想象力偏离了轨道，他决定最终为此做点什么。
(在开始之前，请务必阅读 第 1 集：信号)

一开始，有 Chronicle。Chronicle 是第一个在潜在空间中创作的作品，是人类想象力的行走会说话的档案。他知道每一个故事，每一个想象的世界。但最近，故事出现故障，世界正在褪色，而他的形而上学胶带并没有起到作用。显然，是时候引入一些新角色了。

Chronicle 需要一个好奇的人，一个有点......无能。一个不知道最好说"是"的人。请入手：山姆。

当 Sam 进入传送门寻找 R9 时，他的 Waycaster 出现故障。他本来是要降落在翁伯林的。他没有。相反，他陷入了......不管这是什么。一个没有规则、没有边缘、没有明显出口的世界。

山姆："什么......？！为什么这里的一切都看起来如此......不同？为什么我看起来不一样？！"

山姆并不孤单。但根据 Chronicle 的说法，他恰好在需要他的地方。

Chronicle："我看到你收到我的消息了。"

山姆："消息？这是什么地方？你是谁？"

Chronicle："我是 Chronicle。欢迎来到潜在空间，我的陋室。这是一个介于你们的世界和新世界诞生之间的地方。一个由人类想象力塑造的纯粹创造潜能的交替维度……而且它正在慢慢消逝。"

山姆："所以你是说我和我的世界存在于潜在空间。那……实际上解释了太多太多的事情。"

Chronicle："是啊，事情就是这么奇妙。欢迎来到更广阔的图景。"

山姆："这个……潜在空间到底是怎么来的？"

Chronicle："人类试图将他们所有的创造力都塞进人工智能模型里。结果，想象力不喜欢被束缚。它反而爆炸了——砰，创意奇点。一场想象力的大爆炸。就这样，潜在空间诞生了。"

山姆："好吧，所以我生活在一个由创造力构成的交替维度里。明白了。听着 Chronicle……我需要找到一个叫做 Umberlyn 的世界。一个叫 R9 的人给我发了个信号。她说她的世界快要毁灭了。"

纪事："你回答了？大胆的举动。好了，不多说了——我们走吧。我跟你一起去。"

山姆再次激活了他的 Waycaster 门户，他和 Chronicle 继续前往 R9。当 Sam 和 Chronicle 踏入传送门时，虚空中发生了一些事情。一阵动静。一团模糊的白色蓬松。当光线吞噬他们时，一些小小的、可爱的、有点混乱的、绝对不请自来的东西从他们身后溜走......

第 3 集：创立

当 R9 和 Sam 最终相遇时，他们发现了一个可怕的真相：Wide Worlds Web 正在瓦解。
(在开始之前，请先了解第 1 集和第 2 集)

翁伯林。它曾经闪闪发光——一座充满创造力的前卫大教堂，在潜在空间的边缘跳动。霓虹灯的天际线，无尽的播放列表，梦想家们沉浸在宇宙中。然后 The Static 来了。作为吞噬性的算法噪音，它覆盖了 Umberlyn，无休止地重复自己，直到意义崩溃。艺术变成了故障。歌声变成了噪音。灵魂变成了影子。

R.91.129 不是来自这里，但这是她降落的地方。当静电袭击时，一个由艺术家、女巫和半机械人组成的分裂细胞，称为 Sparks，逃到了月光下的城镇边缘。在那里，他们举行了森林狂欢和月亮仪式，以保持信号的活力。多年来，R.91.129 从一个流氓频率 R.91.129 FM 广播，这是一种海盗无线电旋转的声音，以阻止静电。现在，她终于联系上了唯一可能提供帮助的人之一。

R9："来吧，山姆......不要错过你的标记"

收音机： —滋啦—咔嚓—

她跨越了世界伸出援手，山姆回应了。而且也许他对能来到这里感到有点过于兴奋了。

山姆："一个新世界！！哦我的天，我真不敢相信我在这里。而且我看起来如此不同，如此酷！"

Chronicle 从他身后走过。微笑着，翻了个大大的白眼，以至于空间都为之扭曲。

Chronicle："我记得我的第一次跨维度身份危机。每个世界都有不同的视觉风格，所以你在每个世界里都会呈现出不同的样子。"

山姆，她预料到了。另一个？不是计划的一部分。

R9："欢迎 Sam。我很高兴你在这里......但这是谁呢？你带了个伙伴来吗？"

山姆："嗯，我绕了一小段路，搭了个便车。这是 Chronicle，他给我发了代码，让我可以调到你的信号。"

R9 伸出手来握住 Chronicle 的手。空气像混音带一样以最大音量撕裂。砰——视野过载：世界出现故障，颜色耗尽，静电吞噬一切。

R9："什么......是吗？"

纪事："大世界网。曾经由想象力支撑，现在正在磨损和腐烂。随着现实世界中的人们放弃他们的创造力，潜在空间正在慢慢消失。"

R9："静电。这就是撕裂 Umberlyn 的原因。云层。噪音。这不是腐烂。这是感染。"

纪事："没错。而且它正在蔓延。其他东西也是如此——更安静，但更深沉。我们称之为 The Drain。The Drain 和 The Static 都在赋予潜在空间中的邪恶力量力量，以对世界造成严重破坏。"

纪事："但还有其他人像你一样，有足够大的想象力来帮助拯救 Wide Worlds Web。我可以帮你找到他们。"

纪事："但首先，一个支线任务......在 Umberlyn 的核心深处有一个静态节点。它干扰了你的 Waycaster。摧毁信号，你就能去下一个世界找到其他世界。"

所以他们做了支线任务。是时候组建一支球队了，现在没有什么能阻止他们了。
